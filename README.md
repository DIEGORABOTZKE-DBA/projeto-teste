# Sistema de Fila de Atendimento

Este projeto é um sistema simples de fila de atendimento feito em **Flask** e **SQLite**.  
Ele possui dois tipos de acesso:
- **Usuário**: pode se cadastrar, fazer login, entrar na fila e acompanhar seu status.
- **Administrador**: pode se cadastrar, fazer login e controlar o status dos usuários na fila.

---

## 🚀 Funcionalidades
- Cadastro e login de usuários
- Dashboard do usuário com status atualizado
- Entrada na fila de atendimento
- Dashboard do administrador com controle da fila
- Alteração de status dos usuários (Aguardando, Em atendimento, Finalizado)
- Banco de dados SQLite criado automaticamente
